# KIGTTS 安卓版功能介绍

> 面向用户的功能说明
> 适用对象：Kigrumi / Kigurumi 玩家、漫展跑场用户、需要低负担交流与整活辅助的人群

## 1. 软件定位

KIGTTS 是一款面向 Kigrumi 玩家设计的安卓辅助工具。

它的核心目标不是做“纯语音转文字”软件，而是把 **现场交流、气氛互动、快捷展示、整活演出** 这些需求整合到一套工具里，尽量减少频繁切 App、现场打字慢、穿戴状态下不方便说话等问题。

比较适合这些场景：

- 穿戴头套、面具、妆造后不方便开口说话
- 漫展现场需要快速和路人、同好、摄影沟通
- 需要活跃气氛、做节目效果、播放固定台词或音效
- 需要快速展示二维码、联系方式、名片信息
- 需要在软件外直接操作常用功能

## 2. 这款软件能做什么

KIGTTS 目前把常用能力拆成几个主页面：

- `便捷字幕`
- `快捷名片`
- `画板`
- `音效板`
- `悬浮窗与热键`
- `语音包`
- `设置`

这些功能可以单独使用，也可以组合使用。

例如：

- 用语音识别把说的话直接上屏成大字幕
- 用快捷文本一键发送常用句子
- 让字幕、TTS、音效板联动
- 用悬浮窗在别的 App 上方直接打开快捷字幕、名片或音效
- 用二维码页面快速处理微信 / QQ / 支付宝等常见扫码场景

## 3. 核心功能说明

### 3.1 便捷字幕

这是最核心的交流页面之一。

主要能力：

- 实时语音识别，并把结果显示成大字幕
- 支持快捷文本，常用句子可一键发送
- 支持手动输入文本
- 可配合 TTS 进行朗读
- 可单独调节字幕缩放与展示方式
- 支持编辑分组、排序、导入预设、导出预设

适合场景：

- 不方便开口时直接把话显示出来
- 需要反复说同样的话，例如“稍等一下”“可以合影吗”“谢谢你”
- 跑场时快速与摄影、路人或同伴沟通

### 3.2 快捷名片

快捷名片用于展示身份、备注、链接和二维码。

主要能力：

- 可制作多张名片并切换展示
- 支持标题、备注、链接、二维码
- 支持横竖屏背景图
- 无背景图时可使用主题色卡片样式
- 支持编辑、排序、快速预览
- 悬浮窗里也可以打开迷你快捷名片

适合场景：

- 展示社交账号、主页、约拍信息
- 展示个人介绍、设定、交流群入口
- 在不方便翻相册或切聊天软件时直接给别人看

### 3.3 画板

画板是一个偏现场交流的辅助页。

主要能力：

- 手写、涂鸦、快速示意
- 适合临时写字、画箭头、做说明

适合场景：

- 嘈杂环境下配合字幕进行补充说明
- 临时画路线、画表情、写关键词整活

### 3.4 音效板

音效板用于播放短音频、制造气氛和节目效果。

主要能力：

- 按分组管理音效
- 支持列表、多列宫格等不同布局
- 支持手动点击播放
- 支持关键词触发音效
- 支持导入多种音频格式并统一转码为 AAC
- 支持音效板预设导入、导出与分享

适合场景：

- 现场整活、接梗、做反应音
- 固定角色效果音、提示音、梗音频
- 配合字幕和 TTS 做完整互动流程

### 3.5 悬浮窗与热键

这是 KIGTTS 和普通字幕类软件差异很大的一块。

主要能力：

- 悬浮窗可在别的 App 上方常驻
- 可直接从悬浮窗打开快捷字幕、快捷名片、画板、音效板等功能
- 支持迷你快捷字幕、迷你快捷名片
- 支持贴边、折叠、展开等交互
- 支持音量键序列热键
- 可把常用第三方应用或快捷方式挂到悬浮窗启动器里

适合场景：

- 在微信、QQ、相机、导航等 App 内快速调用 KIGTTS 功能
- 穿戴状态下尽量减少回主界面的次数
- 一只手快速完成交流、展示、整活操作

### 3.6 语音包

软件默认可用系统 TTS，也支持额外导入语音包。

主要能力：

- 默认使用系统 TTS
- 支持导入 Piper 语音包
- 支持切换、查看、分享、删除语音包

适合场景：

- 想要更稳定地直接使用系统语音
- 想尝试不同风格的本地语音包

### 3.7 二维码与扫码联动

软件内置二维码扫描，并对常见国内使用场景做了联动处理。

主要能力：

- 扫描普通二维码
- 识别到微信二维码时可引导跳转微信处理
- 识别到 QQ 二维码时可跳转 QQ；在特定条件下可直达 QQ 扫一扫
- 识别到支付宝二维码时可跳转支付宝扫一扫

适合场景：

- 现场互加好友、收付款、进群、展示链接
- 避免手动复制粘贴或来回切换多个 App

## 4. 音频与识别能力

KIGTTS 的底层能力不只是一套普通的录音转字幕。

当前重点能力包括：

- 本地实时语音识别
- 本地 TTS 朗读
- AI 语音增强
- 多种语音活动检测模式
- 说话人验证
- 音频测试与设备调试

### 4.1 AI 语音增强

软件支持多种语音增强方式：

- `GTCRN 句级增强`
- `GTCRN 流式增强`
- `DPDFNet2 流式增强`
- `DPDFNet4 流式增强`

适合在漫展、商场、地铁站、场馆走廊这类环境噪声比较多的地方使用。

### 4.2 语音活动检测

软件支持三种语音活动检测模式：

- `阈值式VAD`
- `SileroVAD`
- `混合VAD`

可以根据现场环境、说话音量和响应速度需求进行选择。

### 4.3 说话人验证

如果你担心环境里别人说话也触发你的识别链路，可以开启说话人验证。

主要作用：

- 录入 2 到 3 条本人样本后，仅在更像本人说话时放行
- 减少路人、同伴或环境语音误触发

### 4.4 禁用 TTS

如果你只想上屏，不想发声，也可以直接禁用 TTS。

关闭 TTS 后：

- 文字仍会上屏
- 历史记录仍会保留
- 音效板关键词联动仍可继续使用

这比较适合：

- 需要安静交流
- 不想在当前场合外放声音
- 只把软件当成字幕板或触发器使用

## 5. 预设、导入与分享

KIGTTS 支持把常用内容打包保存，方便备份和迁移。

目前支持：

- 便捷字幕预设导入 / 导出 / 分享
- 音效板预设导入 / 导出 / 分享
- 语音包导入

这意味着你可以：

- 提前做好一套漫展用快捷文本
- 提前准备一套角色音效包
- 在不同设备之间快速迁移配置
- 和同伴共享自己整理好的预设

## 6. 为什么它适合 Kigrumi 玩家

这款软件适合 Kigrumi 玩家，关键不是“能识别语音”，而是它把以下几类需求合在了一起：

- `不方便说话`：可直接上屏大字幕，也可手动打字
- `需要展示身份`：可快速展示名片、二维码、链接
- `需要活跃气氛`：可通过音效板、TTS、快捷文本做互动
- `操作要快`：可通过悬浮窗、热键、迷你控件直接调用
- `现场环境复杂`：支持 AI 语音增强、VAD、说话人验证

简单说，它更像一套 **“面向漫展现场交流的轻量工具箱”**，而不是单一功能 App。

## 7. 推荐使用方式

如果你是第一次使用，比较建议这样上手：

1. 先配置 `便捷字幕`
2. 再准备几条最常用的快捷文本
3. 按需制作 1 到 3 张 `快捷名片`
4. 如果需要整活，再配置 `音效板`
5. 最后再开启 `悬浮窗与热键`

一套比较实用的基础组合是：

- 便捷字幕：负责交流
- 快捷名片：负责展示
- 音效板：负责气氛
- 悬浮窗：负责在软件外快速调用

## 8. 可能需要的权限

根据使用的功能不同，软件可能会请求以下权限：

- 麦克风：用于语音识别、音频测试、说话人验证
- 悬浮窗：用于在其他 App 上方显示悬浮控件
- 文件读取 / 选择：用于导入语音包、音效、预设
- 相机：用于二维码扫描
- 无障碍：仅在部分高级功能下使用，例如更稳定的热键监听、部分直达操作引导

如果你只使用主界面的基础功能，并不一定需要把所有权限都打开。

## 9. 适合什么人，不适合什么人

更适合：

- Kigrumi / Kigurumi 玩家
- 漫展用户
- 需要低门槛现场交流工具的人
- 喜欢准备固定台词、固定音效、展示卡片的人

不太适合：

- 只想要一个极简纯转写工具的人
- 完全不需要悬浮窗、热键、展示与互动功能的人

## 10. 一句话总结

KIGTTS 安卓版是一套围绕 **“不方便说话时怎么更顺手地交流、展示、互动和整活”** 设计的现场辅助工具。

如果你的需求不只是“把声音变成文字”，而是希望把 **字幕、名片、音效、悬浮窗、热键、扫码联动** 都串起来，它会比普通字幕工具更合适。
